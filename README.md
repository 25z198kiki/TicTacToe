# TicTacToe
A simple Tic Tac Toe game using Python backend and HTML/CSS/JS frontend.

## Repository structure

TicTacToe/
├── backend/
│   └── game.py
├── frontend/
│   ├── index.html
│   ├── style.css
│   └── script.js
├── README.md
└── .gitignore

## Run the backend

From the `TicTacToe` folder:

```bash
python backend/game.py
```

## Use the frontend

Open `frontend/index.html` in your web browser.
